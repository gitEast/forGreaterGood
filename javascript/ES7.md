# ES7

## 数组

- includes 方法

  ```js
  const names = ["a", "b"]``;
  console.log(names.includes("a")); // true
  console.log(names.includes("a", 1)); // false 从 索引值1 开始判断
  ```

## 指数运算

```js
const a = 3;
const result = a ** 3; // result = 27
```
